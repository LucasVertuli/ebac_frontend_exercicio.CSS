Aqui iremos aprender um pouco mais sobre como incluir o CSS

O CSS é usado para uma função mais visual, para a criação de layouts e efeitos visuais.

A sua sintaxe é usada na forma de "Seletor"{propriedade: valor;}

  1. Seletor = Qual tag quer mudar (h1,h2,h3) ou qualquer outra tag selecionada.


  2. Propriedade = O que eu quero mudar ( cor de fundo, cor de letra, fonte de letra)


  3. Valor = Coloração ou fonte desejada.


  • Exemplo: O H1 dessa página é a frase "Aqui iremos aprender um pouco mais sobre como incluir o CSS." para mudar a sua cor para vermelho se deve acionar o CSS pelos comandos Style --> Seletor (h1) { color; red.} OBS: O comando Style deve ser acionado na tag head.

  • Exemplo 2: Para mudar a cor de fundo da tela os comandos devem ser Style --> body {backgorund-color: cor desejada}

Podemos também adicionar diretamente na tag, pela seguinte forma:

  1. Identificar qual tag deseja modificar


  2. Propriedade style --> color: cor desejada. OBS: o acionamento deve ocorrer entre a tag e o >
